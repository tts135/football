# 足球数据分析系统使用指南

## 快速开始

### 1. 激活虚拟环境并运行系统

```bash
# 方法1: 使用批处理脚本（推荐）
.\run_in_venv.bat .\src\main.py --mode interactive

# 方法2: 手动激活虚拟环境
e:\pyPreoject\football\venv\Scripts\Activate.ps1
python .\src\main.py --mode interactive
```

### 2. 系统功能演示

系统会自动：
- 加载2021年和2023年的比赛数据（共2177场比赛）
- 处理数据并保存到processed_training_data.csv
- 初始化中超联赛预测器
- 进入交互式预测模式

### 3. 交互式预测示例

在提示符下输入：
```
请输入主队名称: 成都蓉城
请输入客队名称: 北京国安
请输入联赛名称（默认中超）: [直接回车使用默认值]
```

系统将输出类似这样的预测结果：
```
==================================================
比赛预测: 成都蓉城 vs 北京国安
==================================================
预测结果:
主队进球: 1.6
客队进球: 1.2
总进球数: 2.8
主队角球: 5.2
客队角球: 4.8
总角球数: 10.0
主队黄牌: 1.8
客队黄牌: 2.1
总黄牌数: 3.9
==================================================
```

### 4. 其他运行模式

```bash
# 训练模式 - 重新训练联赛基线参数
.\run_in_venv.bat .\src\main.py --mode train

# 预测特定比赛
.\run_in_venv.bat .\src\main.py --mode predict --home-team "成都蓉城" --away-team "北京国安"

# 评估模式 - 5折交叉验证
.\run_in_venv.bat .\src\main.py --mode evaluate --k-folds 5

# 指定不同联赛
.\run_in_venv.bat .\src\main.py --mode interactive --league "英超"
```

### 5. 系统特点

✅ **自动化数据处理**: 自动解析JSON格式的原始数据
✅ **多联赛支持**: 支持中超、意甲、英超、西甲、德甲
✅ **实时预测**: 基于历史数据进行智能预测
✅ **交互友好**: 简单的命令行交互界面
✅ **结果详细**: 提供进球、角球、黄牌等多项预测

### 6. 注意事项

- 首次运行需要一些时间来处理数据
- 确保在虚拟环境中运行（已配置好依赖）
- 支持的球队名称必须与数据中的名称完全一致
- 预测结果基于统计模型，仅供参考

### 7. 故障排除

如果遇到"缺少pandas/numpy依赖"错误：
```bash
# 确保在虚拟环境中安装依赖
.\run_in_venv.bat -m pip install pandas numpy
```
